0x01：项目概述

MarkTrans 是一款使用 Python 开发的基于规则引擎的 Markdown 转换工具。 通过读取预配置在 /res/database.db 中的规则， 将 Markdown 文本转换为用户需要的 HTML 格式。

🌳 AST 转换
通过可配置的规则库（SQLite）将 Markdown 解析为抽象语法树（AST），支持 44+ 种标准及扩展语法。
🎨 HTML 渲染
依据自定义规则将 AST 渲染为符合特定场景需求的 HTML 结构，支持一键切换多种风格模板。
🔧 深度定制
支持各种非标准 Markdown 扩展语法（如 Obsidian Callout），适用于自媒体平台及静态站点生成场景。
🚀
0x02：快速开始

环境要求

Python 3.8+

安装与运行

# 1. 克隆项目
git clone https://github.com/Blue-Seventeen/MarkTrans.git
cd MarkTrans

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动 Web 服务
python app.py
📂
0x03：核心架构

别看文件很多，核心就三个文件： 存放映射规则的 database.db， 读取规则并将 Markdown 转换为 AST 的 markdown_ast_parser.py， 以及将 AST 转换为 HTML 的 ast_html_translator.py。

MarkTrans/
├── res/database.db         # 核心：存放各种转换规则
├── src/main/markdown_ast_parser/ # 核心：Markdown → AST
├── src/main/ast_html_translator/ # 核心：AST → HTML
├── app.py                  # Web 服务入口
└── database.py             # 数据库管理工具
