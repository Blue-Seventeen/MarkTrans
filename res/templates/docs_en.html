0x01: Overview

MarkTrans is a Python-based Markdown converter powered by a rule engine. It reads preconfigured rules from /res/database.db and converts Markdown text into customizable HTML structures.

🌳 AST Parsing
Parse Markdown into AST via configurable SQLite rules, supporting 44+ standard and extended syntax patterns.
🎨 HTML Rendering
Render AST into scenario-oriented HTML with rule templates, and switch styles in one click.
🔧 Deep Customization
Supports non-standard Markdown extensions (e.g., Obsidian Callout) for custom publishing workflows.
🚀
0x02: Quick Start

Requirements

Python 3.8+

Install & Run

# 1. Clone repository
git clone https://github.com/Blue-Seventeen/MarkTrans.git
cd MarkTrans

# 2. Install dependencies
pip install -r requirements.txt

# 3. Start web service
python app.py
📂
0x03: Core Architecture

Although there are many files, three files form the core: database.db for rules, markdown_ast_parser.py for Markdown → AST, and ast_html_translator.py for AST → HTML.

MarkTrans/
├── res/database.db         # Core: conversion rules
├── src/main/markdown_ast_parser/ # Core: Markdown → AST
├── src/main/ast_html_translator/ # Core: AST → HTML
├── app.py                  # Web service entry
└── database.py             # Database utility
